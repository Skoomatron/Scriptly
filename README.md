## Installation

1. Fork the repo
2. Clone it locally with command:
   ```sh
   git clone https://github.com/your_username_/Scriptly.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```

<!-- USAGE EXAMPLES -->
## Usage
1. To get the server running:
   ```sh
   npm run server
   ```
2. To build webpack:
   ```sh
   npm run dev:react
   ```
